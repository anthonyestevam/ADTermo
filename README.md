# 💊 Ad Termo

Aplicativo mobile desenvolvido em **Flutter** para auxiliar no **manuseio seguro de medicamentos termolábeis**, fornecendo orientações rápidas sobre armazenamento, segurança e suporte farmacêutico.

## 🧩 Caso de Uso Real

O **Ad Termo** foi desenvolvido como uma solução mobile sob demanda, destinada ao apoio educacional na área da saúde.

A aplicação foi **comercializada para estudantes universitários**, sendo utilizada como ferramenta auxiliar em apresentações acadêmicas e orientação sobre o correto manejo de medicamentos termolábeis.

O projeto envolveu desde o levantamento de requisitos até a entrega final do aplicativo funcional para os usuários.
---

## 📌 Sobre o Projeto

O **Ad Termo** foi criado para facilitar a consulta de informações críticas relacionadas a medicamentos que exigem controle rigoroso de temperatura.

O aplicativo permite que o usuário identifique rapidamente o medicamento e receba instruções padronizadas de conservação, uso seguro e atendimento profissional.

---

## 🚀 Funcionalidades

✅ Seleção manual do medicamento  
✅ Leitura de QR Code da embalagem  
✅ Exibição da temperatura ideal de armazenamento  
✅ Procedimento guiado em **6 etapas** caso o medicamento esteja fora da temperatura recomendada  
✅ Abas educativas sobre **Uso Seguro**  
✅ Contato direto com farmacêutico  

- 💬 Atendimento via WhatsApp  
- 📞 Ligação telefônica direta

## 📥 Instalação do Aplicativo (APK)

Caso deseje apenas testar o aplicativo, é possível instalar diretamente o APK no dispositivo Android.

### Passos

1. Baixe o arquivo APK: adtermo.apk

2. Envie para o celular ou faça download direto pelo GitHub

3. Ative a opção:
**Configurações → Segurança → Instalar apps desconhecidos**

4. Abra o arquivo `.apk` e finalize a instalação

> ⚠️ Aplicativo disponível atualmente apenas para dispositivos Android.

### Instalação

```bash
git clone https://github.com/anthonyestevam/ADTermo.git
cd ad-termo
flutter pub get
flutter run

### Pré-requisitos
